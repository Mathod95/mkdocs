## Bucket

```yaml linenums="1" title="Bucket with fixed name"
---
apiVersion: s3.aws.m.upbound.io/v1beta1
kind: Bucket
metadata:
  name: app1-bucket-mathod
  namespace: crossplane-system
spec:
  forProvider:
    region: eu-west-3
    tags:
      company: mathod
      project: app1
      environment: production
  providerConfigRef:
    name: default
    kind: ProviderConfig
```

```yaml linenums="1" title="Bucket with autogenerated name prefix"
---
apiVersion: s3.aws.m.upbound.io/v1beta1
kind: Bucket
metadata:
  generateName: app1-bucket-mathod-
  namespace: crossplane-system
spec:
  forProvider:
    region: eu-west-3
    tags:
      company: mathod
      project: app1
      environment: production
  providerConfigRef:
    name: default
    kind: ProviderConfig
```
