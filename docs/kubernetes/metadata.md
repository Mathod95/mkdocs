| Field             | Type                  | Description                                                                                                   | Example                                                                                  |
|--------------------|-----------------------|---------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------|
| **name**           | string                | Unique name for the object within its namespace.                                                              | `name: my-pod`                                                                          |
| **namespace**      | string                | *(Optional)* Namespace the object belongs to. Defaults to “default” if omitted.                               | `namespace: dev`                                                                        |
| **labels**         | map (key-value pairs) | Used to organize, categorize, and filter resources.                                                           | `labels: { app: myapp, tier: backend }`                                                 |
| **annotations**    | map (key-value pairs) | Store arbitrary non-identifying metadata (notes, config, tool info, etc.).                                    | `annotations: { description: "My demo pod", owner: "team-a" }`                          |
| **uid**            | string                | Unique identifier assigned automatically by Kubernetes (don’t set manually).                                  | *(auto-generated)*                                                                      |
| **generation**     | integer               | Indicates the version of the desired state. Set by the system.                                                | *(auto-generated)*                                                                      |
| **creationTimestamp** | string             | Timestamp when the object was created.                                                                        | *(auto-generated)*                                                                      |
| **resourceVersion** | string               | Used for concurrency control.                                                                                 | *(auto-generated)*                                                                      |
| **finalizers**     | list                  | Used to ensure cleanup logic runs before object deletion.                                                     | `finalizers: ["kubernetes.io/pv-protection"]`                                           |
| **ownerReferences** | list                 | Links to parent objects that “own” this resource (used for garbage collection).                               | `ownerReferences: [{ apiVersion: apps/v1, kind: Deployment, name: my-deploy, uid: "..." }]` |
| **managedFields**  | list                  | Tracks which field manager made changes.                                                                      | *(auto-generated)*                                                                      |
